# Armak Technologies

Armak Technologies delivers low-voltage, telecommunications, and enterprise networking infrastructure across Southern California since 1998.

## Project Structure

- `src/`: Source code including modular EJS components.
- `public/`: Static assets (images, logos).
- `ai/`: AI-assisted development state and guidelines.
- `dist/`: Generated production-ready static site.

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
```bash
npm install
```

### Development
Start the local development server with hot-reload:
```bash
npm run dev
```

### Build
Generate the production-ready static site in the `dist/` folder:
```bash
npm run build
```

### Preview
Preview the production build locally:
```bash
npm run preview
```
